# Anvil Memory Bridge Extension

Manifest V3 side-load extension for Doc 41 Phase 3.

Load `extensions/memory-bridge/` with Chrome or Chromium's "Load unpacked".
In options, set the bridge URL and enable injection.

Shortcuts in supported AI chat tabs:

- `Ctrl+Alt+M`: search the local bridge and prepend a compact memory block to the focused editor.
- `Ctrl+Alt+S`: save selected text as a memo through `/memory/save`.

Supported tabs: `claude.ai`, `chatgpt.com`, and `gemini.google.com`.
